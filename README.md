# Student Performance Management System

A web-based Student Performance Management System developed using Flask, MySQL, HTML, and CSS. This application helps manage student records efficiently by providing features to add, search, update, and view student information.

## Features

- Add Student Records
- Search Student Details
- Update Student Information
- View All Students
- User-Friendly Interface
- Database Integration with MySQL

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS

## Installation

1. Clone the repository:

```bash
git clone https://github.com/shreyadhokate2-svg/Student-Performance-Management-System.git
```

2. Navigate to the project directory:

```bash
cd Student-Performance-Management-System
```

3. Install required packages:

```bash
pip install flask
pip install mysql-connector-python
```

4. Configure MySQL database according to your local setup.

5. Run the application:

```bash
python app.py
```

## Usage

After running the application, open your browser and visit:

```text
http://127.0.0.1:5000
```
